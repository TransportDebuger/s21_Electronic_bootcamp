Для решения задачи нужно использовать закон Ома для переменного тока. 
Напряжение на резисторе связано с током через него следующим образом:

U = I \ R

где U – напряжение на резисторе, I – ток через резистор, а R – сопротивление резистора.

Напряжение на выходе резистора равно 1 В:

I = U \ R = 1 \ 1000 = 0.001 А

Теперь рассмотрим входной сигнал от источника импульсного напряжения. Так как форма сигнала синусоидальная, то эффективное значение напряжения будет равно отношению амплитуды к sqrt(2):

Uэфф = A \sqrt(2), 
где A – амплитуда сигнала.

Поскольку эффективное значение напряжения должно быть таким же, как и измеренное напряжение на выходе резистора (то есть 1 В), мы можем найти амплитуду:

A = Uэфф * sqrt(2) = 1 * sqrt(2) = 1.414 вольта

Для выполнения данного задания выставлена амплитуда источника 1.421 В, для удовлетворения условию задачи на симуляции, что несколько отличается от расчетного значения.
Данное отличие связано с особенностями реализации модели вольтметра. Хотя обычно измерительные приборы такого типа показывают среднеквадратичное значение напряжения, т.е. Uэфф
