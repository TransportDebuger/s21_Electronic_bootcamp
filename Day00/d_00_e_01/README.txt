Расчитываем сопротивление цепи для достижения заданных показателей исходя из закона ОМА.

R = V / I = 12V / 0.01A = 1200 Om

Имеем 2 участка соединенных последовательно, при этом участок 2 имеет параллельное подключение резисторов.

Сопротивление последовательно соединенных участков Rt = R1 + R2
Полное сопротивление параллельно соединенных резисторов составляет Rt = 1/( 1/R1 + 1/R2 )
В нашем случае полное сопротивление цепи Rt быть 1200 Ом.

т.е. 1200 = R1 + 1 / ( 1/R2 + 1/R3 ).

Что бы затухание напряжения было равномерным необходимо что бы сопротивления не всех участках цепи было оджимнаковым.
Таким образом R1 = R2 = R3 = x

1200 = x + 1 / ( 1/x + 1/x) = x + 1 / (2 / x) = x + x / 2 = 1.5x

x = 1200 / 1.5 = 800 Ом
