Расчитываем сопротивление нагрузки исходя из имеющихся данных о подаваемом напряжении и потребляемой мощности.
Чтобы самостоятельно рассчитать соотношение Ампер / Ватт или Вт / А, нужно использовать всем известный закон Ома. 
Мощность численно равна произведению тока, протекающего через нагрузку, и приложенного к ней напряжения. 
Определяется одним из трех равенств: P = I * U = R * I² = U²/R.

Рнагр = 100 мВт = 0.1 Вт, U = 5V
I = P / U = 0.1 / 5 = 20mА = 0.02 A
Rнагр = U / I = 5 / 0.02 = 250 Oм

Ток проходящий через источник должен составлять не менее половины, тока проходящего через нагрузку. т.е.
Iист = 40 mА = 0.04 А

Исходя из первого закона Кирхгофа алгебраическая сумма токов притекающих в узел равна сумме токов истекающих из него.
Исходя из этого ток протекающий через резистор R2 должен равнятся току протекающему через нагрузку обозначенную резистором R3,
так как по условию задачи ток протекающий через нагрузку должен составлять не менее половины тока, протекающего через источник.
I2 = Iист / 2 = Iнагр = 0.02 А
Исходя из того, что в параллельном соединении напряжение на всех участках одинаковое, то сопротивление R2 = 250 Oм.

При наличии нескольких сопротивлений подключенных последовательно, падение напряжения на каждом участке согласно закону Ома будет пропорционально сопротивлению.

Расчитываем сопротивление участка с параллельным соединением.
Rпар = 1 / (1 / 250 + 1 / 250) = 250 / 2 = 125 Oм.
С учетом того, что нам известен ток, протекющий через источник расчитаем сопротивление цепи. 12 / 0.04 = 300 Ом.

Соответственно напряжение на резисторе 1 должно составлять 12 - 5 = 7 В

Исходя из этого 7 / 0.04 = 175 Ом.
